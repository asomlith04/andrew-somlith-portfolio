# Andrew Somlith — Portfolio

B2B Digital Marketing Strategist portfolio. Persona 5-inspired design, built with Vite + React + Tailwind.

## Specialties

Marketing Automation · Growth Analytics · AI-Enabled Demand Generation · Conversion-Focused Content

## Stack

- Vite 6 + React 19
- Tailwind CSS 3
- Motion (framer-motion successor)
- GSAP + ScrollTrigger
- React Router v7

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Structure

```
src/
  components/
    Nav.jsx, PageTransition.jsx
    ui/              shadcn-style components (hero variants, animated text)
  pages/
    Hero.jsx         / (home)
    Work.jsx         /work — case studies
    About.jsx        /about
    Contact.jsx      /contact
  lib/utils.js
  index.css          global styles + cinematic background
```
