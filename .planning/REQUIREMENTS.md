# Requirements: FAUNA LUXE

**Defined:** 2026-02-25
**Core Value:** The FAUNA LUXE brand identity (cream & beige quiet luxury meets wild animal print) is what separates a $60 MacBook case from a $12 Amazon case -- the brand IS the product.

## v1 Requirements

### Foundation

- [ ] **FOUND-01**: Shopify store created with Payments enabled (Visa, Mastercard, Shop Pay, Apple Pay)
- [ ] **FOUND-02**: Custom domain configured and connected to store (e.g. faunaluxe.com)
- [x] **FOUND-03**: Guest checkout enabled (Shopify disables this by default -- must be manually turned on)
- [ ] **FOUND-04**: SSL certificate active, store loads on HTTPS

### Product Catalog

- [ ] **PROD-01**: MacBook model variant selector on all products (M2/M3 13", M2/M3 14", M2/M3 16")
- [ ] **PROD-02**: "How to find your MacBook model" guide accessible from product pages
- [ ] **PROD-03**: Leopard print collection page with all compatible model variants
- [ ] **PROD-04**: Snake/python print collection page with all compatible model variants
- [ ] **PROD-05**: Cow print collection page with all compatible model variants
- [ ] **PROD-06**: Editorial product photography for each print in cream & beige aesthetic

### Brand Experience

- [ ] **BRAND-01**: Homepage with editorial hero section -- conveys FAUNA LUXE identity in 3 seconds on mobile
- [ ] **BRAND-02**: About / brand story page explaining the FAUNA LUXE identity and why it exists
- [ ] **BRAND-03**: Collections / lookbook page with curated editorial grid (Pinterest/Instagram aesthetic)
- [ ] **BRAND-04**: FAQ page with dropshipping shipping timelines proactively disclosed
- [ ] **BRAND-05**: Shipping policy page with estimated delivery windows and dropshipping disclosure
- [ ] **BRAND-06**: Returns and refund policy page

### Checkout & Trust

- [ ] **TRUST-01**: Branded order confirmation emails styled in cream & beige aesthetic (not stock Shopify emails)
- [ ] **TRUST-02**: AfterShip branded tracking page so customers can self-serve order status

### Email & Retention (Klaviyo)

- [ ] **EMAIL-01**: Email capture popup live on store with incentive (e.g. 10% off first order)
- [ ] **EMAIL-02**: Welcome email flow -- brand story + best sellers (sends after signup)
- [ ] **EMAIL-03**: Abandoned cart recovery flow -- minimum 2 emails, recovers lost checkouts
- [ ] **EMAIL-04**: Post-purchase thank you flow -- brand reinforcement and review request

### Social & Marketing

- [ ] **SOCL-01**: TikTok pixel installed and verified firing on key events (PageView, AddToCart, Purchase)
- [ ] **SOCL-02**: TikTok for Shopify channel connected and catalog synced
- [ ] **SOCL-03**: Meta pixel installed and verified firing on key events (for Instagram/Facebook ads)
- [ ] **SOCL-04**: Instagram feed embedded on homepage or lookbook page
- [ ] **SOCL-05**: Influencer discount code system -- unique trackable codes per creator

### Fulfillment & Dropshipping

- [ ] **SHIP-01**: Dropshipping app integrated (AutoDS or Zendrop) with supplier connected
- [ ] **SHIP-02**: Variant-to-SKU compatibility matrix documented -- 3 prints x 4 MacBook models = 12 mappings verified
- [ ] **SHIP-03**: End-to-end test order completed -- order placed, tracked, received, case verified on MacBook chassis
- [ ] **SHIP-04**: Supplier packaging audit -- unboxing experience verified against luxury brand positioning

## v2 Requirements

### Trust & Social Proof

- **SPROOF-01**: Photo reviews app installed (Judge.me or Loox) -- 5+ seeded reviews before relaunch
  *Note: Research flagged this as table stakes at $50-80. Deferred by decision, revisit after v1 launch.*

### Growth & Optimization

- **GROW-01**: Post-purchase upsell flow (ReConvert) -- "Complete the look" cross-sell
- **GROW-02**: UGC gallery page -- customer and influencer photos on-site
- **GROW-03**: "As Seen On TikTok" landing page -- captures viral traffic spikes
- **GROW-04**: TikTok Shop in-app checkout integration

### Expanded Catalog

- **CAT-01**: Additional animal prints (zebra, tiger, etc.) added to catalog
- **CAT-02**: Additional MacBook models added as Apple releases new hardware

## Out of Scope

| Feature | Reason |
|---------|--------|
| Custom-coded / headless storefront | Shopify theme sufficient, adds weeks of dev complexity |
| Custom manufacturing or inventory | Dropshipping model only for v1 |
| Mobile app | Web-first, mobile browser optimized |
| Subscription / loyalty program | Premature for v1, revisit at scale |
| Older / discontinued MacBook models | M2/M3 covers ~80% of market, simpler catalog |
| Live chat | Adds complexity, FAQ + email sufficient for v1 |
| Blog / editorial content | Nice to have, not launch blocker |
