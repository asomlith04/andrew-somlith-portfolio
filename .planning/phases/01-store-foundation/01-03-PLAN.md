    Build the custom coming-soon page. Navigate to Shopify Admin > Online Store > Themes > Edit code.

    FIRST: Determine theme architecture:
    - Look in the theme file list for "templates/password.liquid" (older Liquid themes) OR "templates/password.json" (Online Store 2.0 themes)
    - If "password.json" exists: the section file is at sections/main-password.liquid (or similar) — edit that file
    - If "password.liquid" exists: edit that file directly with the full page template

    IMPLEMENTATION — Build the full coming-soon page with these required elements (all locked decisions from CONTEXT.md):

    1. CREAM & BEIGE AESTHETIC (locked):
       - Background: #F5F0E8 (warm cream)
       - Accent/borders: #D4C5A9 (warm beige)
       - Text: #2C2C2C (near-black, not pure black — softer)
       - Secondary text: #6B5E4E (warm brown-grey)
       - Font: Use system serif for headlines (Georgia or serif stack), system sans-serif for body
       - No Shopify default colors — override everything with inline styles

    2. LIFESTYLE MOOD IMAGERY (locked — no product shots):
       - Use a full-width or large hero background/image placeholder
       - Source free editorial lifestyle images from Unsplash (cream desk setups, animal print textures, fashion flatlays)
       - Suggested search terms: "cream desk aesthetic", "beige fashion flatlay", "leopard print texture"
       - Reference image by absolute URL from Unsplash CDN (no download needed): https://images.unsplash.com/[id]?auto=format&fit=crop&w=1200&q=80
       - If Unsplash CDN fails to load in theme, upload image to Shopify Admin > Content > Files first, then use the Shopify CDN URL
       - DO NOT use product silhouettes or MacBook case imagery — lifestyle/mood only

    3. BRAND NAME & TAGLINE:
       - FAUNA LUXE in large serif or elegant sans-serif
       - Tagline: "Where the wild meets the luxe." (or elegant variation — Claude's discretion on exact phrasing)
       - Position above the fold on mobile

    4. COUNTDOWN TIMER (locked — vanilla JS, no library):
       - Implement using the verified pattern from RESEARCH.md
       - Placeholder launch date: 90 days from today (2026-02-25 + 90 days = 2026-05-26T00:00:00)
       - Timer IDs: cd-days, cd-hours, cd-mins, cd-secs
       - Style: four columns with large numbers and unit labels (DAYS, HRS, MIN, SEC)
       - Include a clearly commented constant: // UPDATE THIS DATE when launch date is confirmed
       - Handles expired/negative gracefully (Math.max(0, remaining))

    5. EMAIL CAPTURE FORM (locked — "Get 15% off + shop before everyone else"):
       - Use native Shopify {% form 'customer' %} tag (no app, no Klaviyo — Phase 5 scope)
       - Hidden tag fields: contact[tags] = "prospect, coming-soon"
       - CTA text: "Get 15% Off + Shop Before Everyone Else"
       - Input placeholder: "Your email address"
       - Success state: Show "You're on the list. We'll be in touch." (no page reload in Liquid — the form.posted_successfully? check handles this)
       - Style: understated luxury — thin border input, elegant button in beige with dark text

    6. BRAND STORY SNIPPET (locked):
       - Short 2-3 sentence teaser that establishes FAUNA LUXE identity
       - Suggested copy (Claude's discretion — write this): Something that communicates: fashion-first MacBook case, animal print elevated to quiet luxury, cream & beige meets the wild
       - Example direction (not final copy): "FAUNA LUXE was born from a single idea: your MacBook deserves to look as considered as the rest of your outfit. We design animal print cases in cream and beige tones — not loud, not basic. Just wild, quietly."
       - Positioned below the email capture or as a small section above it

    7. PASSWORD GATE (subtle — not the hero):
       - Use {% form 'storefront_password' %} tag
       - Position subtly — small text at bottom of page: "Have a password? Enter store →" with a minimal inline form
       - Do not make this the visual focus — the teaser IS the page; the password is secondary

    FULL TEMPLATE STRUCTURE:
    ```liquid

    <head>
      <meta charset="utf-8">
      <meta name="viewport" content="width=device-width, initial-scale=1">
      <title>{{ shop.name }} — Coming Soon</title>
      <style>
        /* All CSS inline — no external theme assets load on password page */
        /* ... (full cream/beige styles) ... */
      </style>
    </head>
    <body>
      <!-- Hero: mood image + brand name + tagline -->
      <!-- Countdown timer -->
      <!-- Email capture with 15% off incentive -->
      <!-- Brand story snippet -->
      <!-- Password gate (subtle, bottom of page) -->

      {% form 'customer' %}...{% endform %}
      {% form 'storefront_password' %}...{% endform %}

      <script>
        // Countdown timer — vanilla JS, no dependencies
        // UPDATE THIS DATE when launch date is confirmed:
        const LAUNCH_DATE = new Date('2026-05-26T00:00:00').getTime();

        setInterval(tick, 1000);
        tick();
      </script>
    </body>
    </html>
    ```

    If the theme uses password.json (OS 2.0): adapt the section file instead, but keep {% layout none %} in the JSON template's schema and all CSS/JS inline in the section.

    MOBILE-FIRST: Build for 375px viewport first. Ensure all text is readable, form is usable, and countdown is visible without scrolling on mobile.
