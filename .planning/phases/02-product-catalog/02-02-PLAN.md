  <action>
**Step 1: Create the metafield definition**

Navigate to Shopify Admin > Settings > Custom data > Products > Add definition.
- Name: Compatibility Guide
- Namespace and key: custom.compatibility_guide
- Type: **rich_text** (NOT multi_line_text — multi_line_text does NOT render HTML in collapsible row blocks. This is a confirmed pitfall from research.)
- Description: "How to find your MacBook model — displayed as collapsible accordion on product page"
- Save

**Step 2: Fill the metafield content for all 3 products**

Navigate to each product in Shopify Admin > Products, scroll down to Metafields, find "Compatibility Guide", and enter the following rich text content (identical for all 3 products):

```
How to find your MacBook model

1. Click the Apple menu () in the top-left corner of your screen.
2. Select "About This Mac" — your model name appears at the top.

Look for: MacBook Air (13-inch, M2) or MacBook Pro (14-inch, M3 Pro), etc.

Not sure? Email us at hello@faunaluxe.com and we'll help.
```

Tone: helpful aside, not a warning. Two steps only. Students are tech-literate — per locked decision.

Do this for all 3 products: The Leopard, The Python, The Holstein.

**Step 3: Wire the collapsible row block in the theme editor**

Navigate to Shopify Admin > Online Store > Themes > Customize > select a product page template.

In the product information section:
1. Click "Add block" below the existing blocks
2. Select "Collapsible row" (or "Collapsible tab" depending on theme version)
3. Set heading/title to: "How to find your MacBook model"
4. For the content/row content field: click "Insert dynamic source" (the database icon)
5. Select: Products > Metafields > Compatibility Guide (custom.compatibility_guide)
6. Position this block BELOW the variant selector / buy button area
7. Save

This wires the metafield content to display as a collapsible accordion on EVERY product page that uses this template — all 3 products inherit the block automatically. Content differs per product (though in practice it is identical text).

**IMPORTANT:** The theme currently active may be Dawn or the Shopify default. Build the accordion in whatever theme is currently installed. Phase 3 will migrate to Prestige theme and re-wire the metafield to Prestige's equivalent block. Metafield content is theme-agnostic — only the display block changes.

**Step 1: Install the app**

Navigate to Shopify Admin > Apps > Search Shopify App Store for one of these free-tier back-in-stock apps (in preference order):
1. "Notify! Back in Stock | Restock" by Notify Me (free plan available)
2. "Appikon - Back In Stock" (free plan available)
3. "Swym Back in Stock Alerts" (free plan available)

Install whichever is available on free tier. These apps inject a per-variant email capture form when a variant's inventory is 0 and "Continue selling when out of stock" is unchecked.

**Step 2: Configure the app**

After installation:
1. Enable the app on ALL product pages (not just specific products)
2. Customize the button text to: "Notify Me When Available" (or closest option)
3. Customize the email capture popup/form to match FAUNA LUXE aesthetic if the app supports it:
   - Background: cream/off-white (#FFFDF5 or similar)
   - Button color: warm beige/tan (#C9A96E or similar)
   - Text: "We'll email you when this model is back in stock."
4. Set the notification email template to include "FAUNA LUXE" in the subject line
5. Save configuration

**Step 3: Verify behavior on product page**

Since all variant quantities are set to 0 (from Plan 02-01) and "Continue selling when out of stock" is unchecked, every variant should currently trigger the Notify Me capture. This is expected behavior at this stage — dropshipping stock will be connected in Phase 4.

**NOTE on "Continue selling when out of stock":** Plan 02-01 explicitly set this to unchecked for all variants. Do NOT change it. The back-in-stock app requires inventory tracking ON and quantity at 0 to trigger. In Phase 4, when dropshipping is connected, the app (AutoDS/Zendrop) will manage inventory levels and "Continue selling when out of stock" will be re-evaluated.
